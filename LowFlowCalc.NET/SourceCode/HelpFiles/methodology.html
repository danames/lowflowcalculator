Low Flow Calculator™ Alogrithm and Methodology

Low Flow Calculator™ uses a simple algorithm for computing low (or high) flow with a given return period.  The appriach is as follows:

  1. The n-day period with the lowest average flow (Qni) each year (i) is identified and stored. 
  2. The mean and standard deviation of the natural logs of this data is computed: Mean(ln(Qni)) for all years, i.
  3. An inverse-normal function is used to extract the natural log of the 1/m probability of recurrance streamflow (where m = return period in years.)
  4. An exponent function is used to return the required nQm streamflow value.

Confidence bounds for the nQm streamflow value are estimated as follows:

  1. The data set of yearly average low flows (Qni) is resampled with replacement 1,000 times to produce 1,000 realizations of the yearly average low flows.  
  2. The previous algorithm is used to estimate the nQm streamflow value for each of the 1,000 realizations of the yearly low flow data sets.
  3. These nQm estimates are sorted and ranked, and the values at rank 50 and 950 are reported as the 5% and 95% confidence limits respectively.

Note that because the algorithm used to produce the confidence limits is based on random resampling, it is likely that the confidence limits will vary each time you run the analysis on same data set.
